# Privacy and Security

The data is deeply personal. Default to local-first.

## Recommended workflow

1. Import locally.
2. Summarize locally.
3. Inspect summaries.
4. Only paste selected high-level summaries into ChatGPT for synthesis.

## Redaction backlog

Before any external model use, add optional redaction for:

- email addresses
- phone numbers
- street addresses
- medical details
- company-confidential content
- private names

## Backup

Back up `data/graph.sqlite` and raw exports with encryption. Keep Git for code only.
