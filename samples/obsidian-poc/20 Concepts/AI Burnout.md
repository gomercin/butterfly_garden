# AI Burnout

## Core idea

An AI coding agent can fail in a way that resembles burnout: ignored warnings, accumulated unfinished work, context overload, and conflicting expectations.

## Related

- [[Activation Energy]]
- [[Butterfly Graph]]
- [[Meta Neural Network]]

## Why it matters

This can become a strong metaphor for better human-AI collaboration:

> The agent is not tired, but the system around it creates burnout-shaped failure.

## Possible project angle

A requirements/PO layer for AI coding agents that prevents lost requirements and unfinished implementation threads.
