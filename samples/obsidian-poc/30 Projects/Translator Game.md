# Translator Game

## One-liner

A game where conflict is resolved by identifying missing knowledge and transmitting it between locally rational parties.

## Parent concepts

- [[Locality of Observation]]

## Possible formats

1. Abstract Game-of-Life-like simulation.
2. Narrative RPG about communication.
3. Coaching/training tool.
4. Visual conflict map.

## Weekend build version

A grid with agents holding partial facts. Conflict decreases when compatible missing facts are transmitted.

It combines several recurring threads:

- communication frustration

- systems thinking
