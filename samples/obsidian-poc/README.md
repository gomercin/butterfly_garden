# Butterfly Garden Obsidian PoC

This is a mock Obsidian vault for testing the UI/experience before real exports arrive.

It uses placeholder conversation notes and high-level concepts from the Butterfly Graph idea:

- recurring patterns
- dormant project seeds
- life echoes
- timeline traces
- high-excitement butterflies

Open this folder as a vault in Obsidian.

## Suggested first click

Start with:

`00 Home/Butterfly Garden Home.md`

## Folder meaning

```text
00 Home/          navigation and dashboards
10 Conversations/ mock imported chat/source notes
20 Concepts/      large-neuron concept notes
30 Projects/      possible buildable projects
40 Echoes/        patterns appearing across time
50 Timeline/      life/story timeline views
60 Butterflies/   dormant high-excitement ideas
```

## Plugin assumptions

This vault works without plugins.

If you use Dataview, some notes include optional query blocks that can later become live dashboards.
